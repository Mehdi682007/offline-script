زیرساخت بقا برای اینترنت ملی

این پروژه یک زیرساخت آفلاین/نیمه‌آفلاین برای شرایط قطع اینترنت، تحریم، یا بحران است.
با استفاده از یک «سرور مادر»، تمام سرورهای دیگر می‌توانند بدون GitHub، Docker Hub یا DNS خارجی کار کنند.

امکانات

مدیریت پروژه‌های آفلاین

نصب VPN و Tunnel بدون اینترنت جهانی

بروزرسانی خودکار پروژه‌ها

مناسب سناریوی جنگ و اینترنت ملی

نصب روی سرور مادر


```
bash <(curl -Ls https://raw.githubusercontent.com/Mehdi682007/offline-script/main/installmo.sh)
```
